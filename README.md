# webdevobasic-
Exploring frontend development by building responsive and interactive web projects from scratch. This repository contains my web development learning journey, including projects and exercises built with HTML, CSS, and JavaScript.

## Simple Webpage Example
This project includes a basic webpage demonstrating:

- HTML elements: headings, paragraphs, images, and links
- CSS styling: colors, fonts, spacing, layout, and hover states
- JavaScript interaction: a button that triggers an alert message

### Files
- `index.html` — webpage structure and content
- `styles.css` — styling for layout and design
- `script.js` — JavaScript button interaction

### How to open
Open `index.html` in a browser to view the webpage and click the button to see the alert.
